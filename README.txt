SISTEMA DE REGISTRO DE USUARIOS PARA ACADEMIA

Projeto feito durante o curso de Full-stack da B7web do Bonieky Lacerda

Projeto pessoal feito para simular um sistema de cadastro para academias. 
Nele foi feito um CRUD onde é possível fazer adicionar, ver, atualizar e excluir usuarios.

TECNOLOGIAS UTILIZADAS:
	HTML
	CSS
	JAVASCRIPT
	PHP
	BANCO DE DADOS MYSQL
